# Pasos Realizados

## Paso 1

El desarrollador líder crea un repositorio con una `issue` y el colaborador le hace un `fork`.

## Paso 2

El colaborador hace las modificaciones en una rama de su fork, al hacer un `push` le aparece en su `fork` la opción de hacer una `pull request` .

## Paso 3

Una vez realizada la `pull request` el líder establece un dialogo indicandole de que la pull request está bien y acepta la pull request haciendo un `merge` en la rama `main`.
